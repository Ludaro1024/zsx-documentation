## What is ZSX-UI V2?

ZSX-UI is a customizable UI resource for FiveM servers. It offers a wide range of features and customization options to enhance the user experience. and is designed to be easy to use and integrate with other resources.

it includes:

- Customizable HUD Icons
- Customizable Notifications
- Customizable Progressbar
- Customizable Job / Money / Bank / ID / Online Users Display
- Customizable Vehicle HUD
- Customizable Text UI
- Customizable Pause Menu
- Cinematic mode ( for recording )
- Customizable Progressbar
- AIO Menu (small menu that opens where you can insert your own functions and commands similar to a personalmenu)
- Customizable Pause Menu
- Custom Music (Youtube) in the Pause Menu + Configuration Menu (UI-Settings)
- Ingame Menu to edit the UI per player
- Auto Installer for the UI into ox_lib and Frameworks!
